编辑

编辑

检查安装是否成功

打开命令行窗口（win + R 输入 cmd），输入java -version 查看Java版本号

配置环境变量

右键 此电脑 ——> 属性 ——> 高级系统设置 ——> 环境变量

在系统变量里 新建一个

变量名为 JAVA_HOME

变量值为 D:\Program Files\Java\jdk1.8.0_333 （JDK 的安装位置）

再将 JAVA_HOME 添加到 用户变量中的 Path 里面，即将%JAVA_HOME%\bin添加到 Path 里面

检查是否配置成功
